# Lighthouse Republics Preview

This folder is the public preview package for *Lighthouse Republics: United
States and Venice: A Short Life Together and Forever After*.

Dedicated preview URL:

```text
/lighthouse-republics/preview/
```

The package is intentionally limited. It should contain the cover, full table of
contents, and only the opening chapter or opening few chapters, capped at 10
percent of the book body.

## Included Edition

- South Preview: canonical main edition, available as
  [PDF](https://fl6nu3o2c1oqqnum.public.blob.vercel-storage.com/books/lighthouse-republics/pdf/7cf617ce3b7f4ef8-lighthouse-republics-preview.pdf),
  [EPUB](https://fl6nu3o2c1oqqnum.public.blob.vercel-storage.com/books/lighthouse-republics/epub/aa43a743ae396b1c-lighthouse-republics-preview.epub),
  [online reader](/read/lighthouse-republics/), and
  [chapter reader](/read/lighthouse-republics/chapters/).

East, West, Rosetta, and Rosetta Scored editions are not present in this public
package yet.
