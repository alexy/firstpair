# APC40 MK2 Ableton Live 12 Getting Started

A zero-to-one, complete-newbie guide to live-looping on the Akai APC40 MK2 in
Ableton Live 12. It assumes no prior Ableton or controller experience and walks
from plugging in the APC through building every clip in the piano roll to
recording and exporting a legal practice rebuild of the feel of the Savage /
John E.S "Don't Cry Tonight" remix.

The book is published through the First Pair library as PDF, EPUB, hosted web
readers, and an interactive **Learn** tutorial that animates the APC40 surface
and the Ableton Session grid step by step.
