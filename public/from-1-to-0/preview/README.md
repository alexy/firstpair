# From 1 to 0 — Public Preview

This folder is the public preview package for *From 1 to 0: Notes on Startups,
or How to Lose the Future* (draft edition v0.1.0).

Dedicated preview URL:

```text
/from-1-to-0/preview/
```

The package is intentionally limited. It contains the cover, the epigraphs,
the full plan of the book, the Introduction ("The Silent Evidence"), and the
opening chapter ("The Anti-Naval Ontology of Failure") — just under ten
percent of the book's body text. It does not contain the complete book.
